# 项目简历总结（中文）

## 简历条目（LaTeX `\item`）

```latex
\item 现有基于多模态大模型的VLN方法多将导航建模为离散动作分类，缺乏对连续运动学和三维空间结构的显式刻画，难以输出平滑可执行的轨迹。为此，我们提出一种几何感知的连续轨迹VLN系统：以Qwen3.5-VL多模态大模型为基座，将其最末层隐状态作为统一的视觉-语义-空间条件，驱动连续轨迹生成。我们首先引入VGGT-Omega视觉几何基座模型作为空间强迫（Spatial Forcing）教师，在仅RGB输入下将Qwen中间层特征与三维几何目标进行跨层对齐，作为训练期辅助监督显式注入空间结构理解，且不引入任何推理开销；随后我们设计了一套可插拔的连续轨迹解码接口，在同一基座之上统一支持MLP直接回归与基于流匹配（Flow Matching）的扩散式轨迹头（DiT、NextDiT、OmniNav式航点头），通过预测速度场并以少步Euler积分生成自我中心连续轨迹$[x,y,\theta]$；同时引入可学习的停止头与滚动时域重规划机制，实现端到端的到达判别与闭环执行。大量实验表明，我们的方法在连续环境VLN基准上以纯RGB输入显著提升了轨迹质量与导航成功率，并在多种轨迹生成范式下展现出一致的泛化能力和强指令跟随能力。项目代码：\url{https://github.com/LeonGoretzkatju/uni-vln-joynav}。
```

## 说明

- **结果句相对克制**："显著提升了轨迹质量与导航成功率"。当前仓库仅有 20-step 的 smoke run（SR≈0，仅作流程验证）。正式实验完成后，建议替换为具体数字，例如"在 R2R/RxR 连续环境 val-unseen 上达到 SR xx%、SPL xx%，相比基线提升 xx 个百分点"，量化指标在简历中更有说服力。
- **未套用的内容**：范例中"单目深度估计大模型把多时刻单目提升到 3D / ViT 编码 / 自适应分辨率·Pose-Free·Occupancy-Aware 剪枝"等细节属于另一套系统，本项目并无这些模块，故未纳入。本条目忠实对应本仓库这套方案（Qwen3.5-VL + 空间强迫 + 四种可插拔轨迹头）。
